# Visualizador Atómico

Aplicación en HTML para visualizar los átomos, orbitales, isótopos, aniones, cationes, etc.

Mi hermana tenía examen de física y química de 4ºESO y le entraraban los modelos atómicos, configuración electrónica, etc. 

Es un simulador bastante sencillo en el que se representan los 118 elementos de la tabla periódica en 3D y se puede elegir entre el modelo cuántico o el modelo de Bohr. Si se elige el cuántico, se puede ver la configuración electrónica, los orbitales y una representación de dónde podrían estar los orbitales de cada electrón (aunque es probabilístico, no determinista). 

Como mejoras futuras se podría cambiar el selector de elemento por una tabla periódica y hacer una simulación de cómo enlazan distintos elementos, los tipos de enlace etc., lo dejo por aquí por si alguien tiene ganas de programar. 

Por supuesto, no tiene ninguna validez para uso profesional, pero quizás podría ser útil como rescurso educativo.
