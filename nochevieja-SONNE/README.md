# Nochevieja R/V SONNE

Aplicación en HTML para campaña oceanográfica con temática de biología marina/meiofauna. 

Esta aplicación fue creada para tomarse las uvas en Nochevieja (tradición española) en el buque oceanográfico alemán R/V SONNE, campaña SO317 (Mangan 2026). Simplemente es una cuenta atrás de las 12 uvas, un AUV va muestreando 12 individuos de la meiofauna hasta que muestrea todos y empieza el año nuevo. 

Cuando le das al botón de empezar o al espacio, empieza una cuenta atrás de 5 segundos, y cuando llega el año nuevo hay fuegos artificiales de meiofauna. 

Nos lo pasamos bien.
